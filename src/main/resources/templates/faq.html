How is the Fold Change calculated

If data and background inside counts are below .5% the value is set to 0.0

If one of the counts is 0 the value is set to POSITIVE_INFINITY

In all other cases the inside and outside counts of data and background are divided and the absolute log2 is returned.

If a scored track is set as covariant the P-value is not 1

In contrast to basic inside/outside tracks the background model creation for scored tracks works with an expectancy value. Thus the counts to do excacly match.

Still the track is above the 5% significant level in most cases.

Why is contigs the most significant track?

Background positions are set random. Thus they tend be in parts of the genome were no virus can integrate or no integration can be sequenced. Set the 'contigs' track as covariant to adjust the background model.

What does the influence factor change?

The influence factor changes the factor by which a background model with one or more covariants is influenced by the covariants. A value close to 1 sets the random position with the same expectancy value as the covariants have.

Close to 0 the background positions are normaly distributed as if there were no covariants.

How can I download the background positions?

After a run 'Background positions' under 'Information' in the left bar is a link leading to the file.
